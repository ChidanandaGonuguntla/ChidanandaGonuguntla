## 🌟 Featured: SentryGate — Agentic DevSecOps

> **A Maven plugin that turns security scanning into an autonomous Observe → Reason → Act loop.**

🔍 **Observe** — Runs Nexus IQ, Fortify SCA & SonarQube scans locally
🧠 **Reason** — Sends findings to Anthropic Claude with full code context
🛠️ **Act** — Generates prioritized, exploit-aware remediation suggestions
🔁 **Iterate** — Developers fix and re-scan before pushing — no Jenkins roundtrip

**Stack:** Java 21 · Maven Plugin API · Anthropic Claude · Spring AI · MCP-ready architecture
